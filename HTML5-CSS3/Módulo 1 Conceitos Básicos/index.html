O que é uma TAG?

A TAG, assim como as palavras chaves, são informações específicas que irão classificar os dados obtidos na internet ajudando a encontrar páginas na web dentre outros temas de outros lugares que estejam ligados as informações.
As tags são usadas para informar ao navegador a estrutura do site

Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento.
